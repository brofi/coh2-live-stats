# CoH2LiveStats

## Build

Install `build`:

```console
$ pip install build
```

Run `build`:

```console
$ python -m build
```

## Install

```console
$ pip install .\dist\coh2_live_stats-0.0.1-py3-none-any.whl
```

## Run

```console
$ python -m coh2_live_stats
```

or

```console
$ coh2livestats.exe
```
